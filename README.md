# Sales & Customer Behavior Analysis

## 📌 Project Overview
This project analyzes complex sales records to uncover actionable insights regarding customer behavior and top revenue drivers using advanced SQL.

## 🛠️ Tools & Techniques Used
* **Database Management System:** SQL Server (Azure Data Studio)
* **Database:** AdventureWorks
* **Techniques:** Complex `JOIN`s across multiple schemas, Data Aggregation (`SUM`), Grouping, and Performance Sorting.

## 💡 Key Business Insights
**Who are our most valuable customers?** By analyzing sales order headers and joining them with customer demographics, I successfully identified the top 10 customers by total lifetime revenue, providing a target list for premium marketing campaigns or loyalty programs.
